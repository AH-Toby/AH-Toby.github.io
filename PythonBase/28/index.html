Python函数(二)

函数的参数和返回值

Posted by     "Toby" on Sunday, June 7, 2020

一.什么是函数的参数

所谓函数的参数就是在定义函数的时候可以让函数接收数据的占位符

二.参数的种类

在Python中参数的种类有形式参数(形参),缺省参数,不定长参数

1.形式参数(形参)

作用: 也称为普通参数或位置参数，是在函数定义中声明的参数。它们用于接收函数调用时传递的值，是函数定义的一部分

结构:

def 函数名(参数名):
    代码（代码中可以使用参数）

demo:

def add2num(a, b):
    c = a+b
    print(c)

2.缺省参数(默认参数)

作用: 在函数定义中赋予了默认值的参数，如果函数调用时没有提供对应的实参，将会使用默认值。

结构:

def 函数名(参数名=默认值):
    代码（代码中可以使用参数）

demo:

def printinfo(name, age=35):
   # 打印任何传入的字符串
   print(f"name:{name}")
   print(f"age:{age}")

3.不定长参数

不定长参数也可以叫可变长参数，根据功能还可以分为不定长位置参数和不定长长关键词参数，以下是具体的分类情况

3.1.不定长位置参数(args)

作用: 在函数定义时，通过在形参前加一个星号 *，表示接受任意数量的位置参数，并将它们存储在一个元组中

结构:

def 函数名(*形参):
    执行功能
    return 返回值

demo:

def sum_all(*args):
    total = 0
    for num in args:
        total += num
    return total

3.2.不定长关键词参数

作用: 在函数定义时，通过在形参前加两个星号 **，表示接受任意数量的关键字参数，并将它们存储在一个字典中

结构:

def 函数名(**形参):
    执行功能
    return 返回值

demo:

def print_info(**kwargs):
    for key, value in kwargs.items():
        print(key + ": " + value)

4.参数的位置顺序

函数定义时参数的位置前后顺序为: 形参 -> 缺省参数 -> 不定长位置参数 -> 不定长关键词参数

结构:

def 函数名(形参，*形参，**形参):
    执行功能
    return 返回值

demo:

def fun(a, b, *args, **kwargs):
     """可变参数演示示例"""
    print("a =%d" % a)
    print("b =%d" % b)
    print("args:")
    print(args)
    print("kwargs: ")
    for key, value in kwargs.items(): #遍历字典
        print("key=%s" % value) #输出字典

fun(1,2,3,4,5,m=6,n=7,p=8)#调用函数

三.实参

作用: 是函数调用时传递给函数的具体值或变量。实参是实际用于执行函数操作的值，它们被传递给函数，并赋值给函数定义中对应的形式参数。当函数被调用时，实参可以是任何合法的Python表达式，包括变量、常量、表达式等。实参的数量和类型必须与函数定义中的形式参数匹配，否则可能会引发错误。

结构:

函数名(实参)

demo:

def add2num(a, b):
    c = a+b
    print(c)

# 调用函数并传递实参
add2num(a=110, b=22)

四.调用带参数的函数

1.格式1

函数名(参数名=数据)

demo:

add2num(a=110, b=22)

2.格式2

函数名(数据)  # 必须是参数名有几个数据就有几个，且位置对应

demo:

add2num(1,2)

3.函数调用参数位置顺序

函数调用时函数中实参位置必须和定义时位置一致，否则会出现传递错误的情况

demo:

def fun(a, b, *args, **kwargs):
     """可变参数演示示例"""
    print(f"a = {a}")
    print(f"b = {b}")
    print(f"args:{args}")
    print(f"kwargs:{kwargs}")

fun(1,2,3,4,5,m=6,n=7,p=8)#调用函数

运行结果:

a = 1
b = 2
args:(3, 4, 5)
kwargs:{'m': 6, 'n': 7, 'p': 8}

五.什么是函数返回值

所谓 返回值 ,就是程序中函数完成一件事情后,最后给调用者的结果

结构:

def 函数名(参数名/无参数名):
    “”“注释”“”
    代码
    return 返回值

demo:

def add2num(a, b):
    c = a+b
    return c

六.保存函数的返回值

所谓的保存其实就是将返回值给予给一个新的变量

demo:

#定义函数
def add2num(a, b):
    return a+b

#调用函数，顺便保存函数的返回值
result = add2num(100,98)

#因为result已经保存了add2num的返回值，所以接下来就可以使用了
print(result)
