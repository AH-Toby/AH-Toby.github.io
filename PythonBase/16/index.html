Python基本语法(六)

条件判断语法补充知识

Posted by     "Toby" on Monday, June 1, 2020

一.区间比较

a = 10
1<a<20  # True
11<a<20  # False

不用写and直接在数据区间进行比较

二.三元运算符

所谓三元运算是指：当条件成立时把 值1 赋值给变量，条件不成立时把 值2 赋值给变量。

格式：

变量 = 值1 if 条件 else 值2

demo:

a = 10
b = 20
c = a if a>b else b
print(c)  # 条件不成立，把 b 的值赋给 c，也就是 c 的值为 20

三.数字的逻辑运算

a = 0
b = 1
c = 3

# and 运算，只要有一个值为 0 则结果为0，否则结果为最后一个非 0 数字
a and b  # 0
b and a  # 0
c and b  # 1
b and c  # 3

# or 运算，只有所有值为 0 结果才为0，否则结果为第一个非 0 数字
a or a  # 0
a or b  # 1
b or a  # 1
c or b  # 3
b or c  # 1

四.is的用法

在Python中，is是一个用于比较对象标识的操作符。它用于检查两个对象是否引用同一块内存空间，而不是值是否相等。

结构:

if xx == True:
  xxxx

上述写法也对但是不符合pep8标准

最好按照下列结构来写：

if xx is True:
  xxxx
