函数的嵌套调用

Posted by     "Toby" on Sunday, June 7, 2020

一.函数的嵌套调用以及实例

作用: 函数的嵌套调用是指在一个函数的函数体内调用另一个函数。这种技术允许你构建更复杂的函数和程序结构，通过将功能分解为更小的部分来提高代码的可读性和可维护性。

结构:

def 函数1(参数):
    函数2(实参)
    ...

def 函数2(实参):
    函数n(实参)
    ...

demo:

def greet(name):
    return "Hello, " + name + "!"

def welcome():
    username = input("Please enter your name: ")
    greeting_message = greet(username)
    print(greeting_message)

# 调用嵌套的函数
welcome()

运行结果:

Please enter your name:toby
Hello, toby!

在这个例子中，welcome() 函数调用了 greet() 函数。当 welcome() 被调用时，它要求用户输入他们的名字，然后调用 greet() 函数来生成问候消息，并将该消息打印出来。

小总结:

1.一个函数里面又调用了另外一个函数，这就是所谓的函数嵌套调用

2.如果函数A中，调用了另外一个函数B，那么先把函数B中的任务都执行完毕之后才会回到上次 函数A执行的位置

3.函数的嵌套调用可以有多层嵌套，也就是在一个函数内部调用另一个函数，而被调用的函数又可以调用其他函数，以此类推。这种嵌套调用的设计允许你以逻辑上的模块化方式组织代码，提高代码的可重用性和可维护性。
