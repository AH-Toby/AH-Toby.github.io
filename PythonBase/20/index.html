循环语句for

Posted by     "Toby" on Tuesday, June 2, 2020

一.什么是for循环

像while循环一样，for可以完成循环的功能。 在Python中 for循环可以遍历任何序列的项目，如一个列表或者一个字符串等。

二.for循环的结构

for 临时变量 in 列表或者字符串等可迭代对象:
    循环满足条件时执行的代码

demo1:

name = 'helllo world!'

for x in name:
    print(x)

demo2:

name = 'helllo world!'
for x in name:
    if x == 'e':
        print("不能使用字符 e")
    else:
        print(x)

demo3:

for 循环的 else 语句只有当循环正常结束才会执行，也就是没有被 break 打断才会执行

name = 'helllo world!'
for x in name:
    if x == 'e':
        print("不能使用字符 e")
        break
    else:
        print(x)
else:
    print('没有出现禁止使用的字符')

运行结果如下:

h
不能使用字符 e
