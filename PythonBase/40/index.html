Python面向对象编程基础(一)

Python面向对象介绍

Posted by     "Toby" on Thursday, June 11, 2020

一.什么是面向对象？

生活中我们用洗衣机洗衣服不需要考虑洗衣机的构造和制造我们只需要直接把衣服丢进去即可，这种 让其他专业的事物去做的思想 就是面向对象，每一步我们都不需要亲力亲为，只需要关注结果就行。在编程中面向对象编程思想将数据与函数绑定到一起分类进行封装，每个程序员只要负责分配给自己的分类，这样能够更快速的开发程序，减少了重复代码。

二.面向过程和面向对象对比

面向过程: 根据业务逻辑从上到下写代码，每一步都是自己做。

面向对象: 将数据与函数绑定到一起，分类进行 封装，每个程序员只要负责分配给自己的分类，这样能够更快速的开发程序，减少了重复代码，每一步都照专业的人做

三.面向对象和面向对象编程

1.面向对象(Object Oriented,简称:OO)

至今还没有统一的概念，我这里把它定义为:

按人们认识客观世界的系统思维方式，采用基于对象(实体)的概念建立模型，模拟客观世界分析、设 计、实现软件的办法。

2.面向对象编程(Object Oriented Programming,简称OOP)

是一种解决软件复用的设计和编程方法，简单说就是一种编程范式。

其中的核心概念是 对象 、 类 和 方法 。在面向对象编程中，程序的主要构建单元是对象，这些对象是对现实世界中的实体或概念的抽象表示。每个对象都有自己的 属性 (称为数据成员或字段)和行为(称为方法或函数)。

四.类和对象

对象是面向对象编程的核心，在使用对象的过程中，为了将具有共同特征和行为的一组对象抽象定义，提出了另外一个新的概念: 类

类就相当于制造飞机时的图纸，用它来进行创建的飞机就相当于对象

1.类

概念: 具有相似内部状态和运动规律的实体的集合(或统称为抽象)，简单说就是具有相同属性和行为事物的统称。类是对象的模板或蓝图，用于创建特定类型的对象。类定义了对象的属性和方法。一个类可以找到多个对象。

2.对象

概念: 某一个具体事物的存在,在现实世界中可以是看得见摸得着的。可以是直接使用的。对象是类的实例。它是内存中的实体，具有特定的属性和方法。

3.类和对象的关系

简单举例来说：类和对象的关系好比就是做饼的模子和饼之间的关系，通过模子来制造饼。类就是创建对象的模板

具体关系:

  • 类是对象的蓝图或模板: 类定义了对象的属性和行为(方法)。对象是根据类创建的具体实例
  • 类是抽象的,对象是具体的: 类描述了一组共性，比如"人类"这个类。对象是具体存在的个体,比如具体的"张三"这个人
  • 对象通过类实例化创建: 使用类似 obj = MyClass() 的语句,根据类实例化创建对象。对象拥有该类定义的所有属性和方法
  • 类是逻辑上的概念,对象是内存中的实体: 类只是一个逻辑概念,直到实例化后才真正存在。对象在内存中占有空间,可访问其属性和方法
  • 一个类可以有多个对象实例: 同一个类可以创建出多个不同的对象实例。不同对象拥有独立的属性值,但方法是共享的
  • 通过对象访问类的属性和方法: 使用 object.attribute 访问对象的属性。使用 object.method() 调用对象的方法

4.类的构成

类(Class) 由以下部分构成

  • 类名(Class Name): 是在定义类时指定的，通常使用大驼峰命名法，例如 Car。

  • 属性(Attributes): 属性是类中的变量，用于存储对象的状态或数据。在Python中，属性可以是类变量（被所有实例共享）或实例变量（每个实例拥有自己的副本）。

  • 方法(Methods): 方法是类中的函数，用于定义对象的行为。类中的方法可以访问并操作对象的属性，也可以执行其他操作。

  • 构造函数(Constructor): 构造函数是一种特殊的方法，它在创建类的实例时自动调用。在Python中，构造函数的名称是 __init__()，它用于初始化对象的属性。

  • 特殊方法(Special Methods): Python还提供了一些特殊方法，也称为魔术方法或双下划线方法（dunder methods），用于实现类的特殊行为，例如 __str__() 方法用于定义对象的字符串表示形式，__len__() 方法用于定义对象的长度。

  • self参数: 在类的方法中，第一个参数通常被命名为self，它表示对象本身。通过self参数，方法可以访问并操作对象的属性和方法。

demo:

class Car:
    # 构造函数
    def __init__(self, make, model):
        self.make = make  # 实例变量
        self.model = model  # 实例变量

    # 实例方法
    def drive(self):
        print(f"{self.make} {self.model} is driving")

    # 特殊方法
    def __str__(self):
        return f"{self.make} {self.model}"

# 创建Car类的实例
car = Car("Toyota", "Corolla")

# 访问实例变量
print(car.make)  # 输出：Toyota

# 调用实例方法
car.drive()  # 输出：Toyota Corolla is driving

# 调用特殊方法
print(car)  # 输出：Toyota Corolla

5.如何把日常生活中的事物抽象成程序中的类?

把日常生活中的事物抽象成程序中的类需要考虑事物的属性和行为，并将其转化为类的属性和方法。

方法: 一般名词都是类(名词提炼法)

人类:

  • 属性：名字、年龄、身高、体重、发型等。
  • 方法：跑、走、思考、吃饭等。

汽车:

  • 属性：品牌、型号、颜色、速度、油量等。
  • 方法：启动、加速、刹车、加油等。

手机:

  • 属性：品牌、型号、颜色、屏幕尺寸、电池电量等。
  • 方法：打电话、发短信、拍照、充电等。

动物:

  • 属性：种类、年龄、体重、颜色等。
  • 方法：发出声音、移动、进食、休息等。
