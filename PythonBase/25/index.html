Python基本数据类型(五)

字典-dict

Posted by     "Toby" on Wednesday, June 3, 2020

一.什么是字典

作用: 由键-值对组成的数据, 用于以键值对的方式存储数据

格式:

{key:value}

说明:

1.字典和列表一样，也能够存储多个数据

2.列表中找某个元素时，是根据下标进行的，字典中找某个元素时，是根据key

3.字典的每个元素由2部分组成，键:值。

4.字典中若访问不存在的键，则会报错，使用字典名.get(key)访问，若访问不存在的键不会报错会返回none

二.字典中的常见操作

1.增

如果在使用 变量名['键'] = 数据 时，这个 键 在字典中，不存在，那么就会新增这个元素

demo:

info = {'name':'班长', 'sex':'f', 'address':'地球亚洲中国北京'}

newId = int(input('请输入新的学号'))

info['id'] = newId

print('添加之后的id为:%d'%info['id'])

运行结果:

请输入新的学号188
添加之后的id为: 188

2.删

2.1.del

删除指定的元素

结构:

del mydic[key]

demo:

info = {'name':'班长', 'sex':'f', 'address':'地球亚洲中国北京'}
print('删除前,%s'%info['name'])
del info['name']
print('删除后,%s'%info['name'])  ## 删除后不能访问会报错

运行结果:

KeyError: 'name' 
删除前,班长

2.2.pop

取出指定键的元素并删除这个元素

结构:

mydic.pop(key)

demo:

info = {'name': 'monitor', 'sex': 'f', 'address': 'China'}
print(f"取出name:{info.pop('name')}后，字典:{info}")

运行结果:

取出name:monitor后，字典:{'sex': 'f', 'address': 'China'}

2.2.clear

清空整个字典

结构:

mydic.clear()

demo:

info = {'name':'monitor', 'sex':'f', 'address':'China'}

print('清空前,%s'%info)

info.clear()

print('清空后,%s'%info)

运行结果:

{}

3.改

字典的每个元素中的数据是可以修改的，只要通过key找到，即可修改

demo:

info = {'name':'班长', 'id':100, 'sex':'f', 'address':'地球亚洲中国北京'}

newId = input('请输入新的学号')

info['id'] = int(newId)

print('修改之后的id为:%d'%info['id'])

运行结果:

请输入新的学号12
修改之后的id为:12

4.查

除了使用key查找数据，还可以使用get来获取数据

结构:

mydict[key]  # 方法一
mydict.get(key)

demo:

info = {'name':'吴彦祖','age':18}

print(info['age']) # 获取年龄

# print(info['sex']) # 获取不存在的key，会发生异常

print(info.get('sex')) # 获取不存在的key，获取到空的内容，不会出现异常

运行结果:

18
None

三.字典的常见操作2

1.len

作用: 测量字典中，键值对的个数

结构:

len(mydic)

demo:

info = {"name":"lili", "age":18, "tel":123456789}

print(len(info))

运行结果:

3

2.keys

作用: 返回一个包含字典所有KEY的列表

结构:

mydic.keys()

demo:

info = {"name":"lili", "age":18, "tel":123456789}

print(info.keys())

运行结果:

dict_keys(['name', 'age', 'tel'])

3.values

作用: 返回一个包含字典所有元素的列表

结构:

mydic.values()

demo:

info = {"name":"lili", "age":18, "tel":123456789}

print(info.values())

运行结果:

dict_values(['lili', 18, 123456789])

4.items

作用: 返回一个包含所有（键，值）元祖的列表

结构：

mydic.items()

demo:

info = {"name":"lili", "age":18, "tel":123456789}

print(info.items())

运行结果:

dict_items([('name', 'lili'), ('age', 18), ('tel', 123456789)])

5.has_key

如果键在字典中，返回True，否则返回False
