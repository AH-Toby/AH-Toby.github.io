Python面向对象编程基础(二)

类和对象的创建和使用

Posted by     "Toby" on Thursday, June 11, 2020

一.类的定义

1.定义类的结构

结构:

class 类名(object):
    # 属性定义
    属性1 = value1

    # 方法定义
    def 方法(self, parameters):
        # 方法实现
        pass

demo:

class Person(object):
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        print(f"Hello, my name is {self.name} and I am {self.age} years old.")

# 创建一个 Person 对象
person1 = Person("Alice", 30)

# 调用对象的方法
person1.greet()  # 输出：Hello, my name is Alice and I am 30 years old.

2.新式类和旧式类

在Python中，新式类和旧式类是两种不同的类的定义方式，主要区别在于它们的继承方式、方法解析顺序（Method Resolution Order，简称 MRO）、属性查找方式等。

Python 2.x 中引入了新式类，Python 3.x 中只有新式类。

新式类(New-style Classes):

  • 继承自 object 类或其子类的类被认为是新式类。
  • 拥有更好的多重继承支持，采用 C3 算法计算 MRO。
  • 在 Python 3.x 中，所有的类都是新式类。

旧式类(Old-style Classes):

  • 不继承自 object 类的类被认为是旧式类。
  • 继承链的搜索顺序是深度优先搜索，而不是 C3 算法计算的 MRO。
  • 在 Python 2.x 中，如果不显式地继承自 object 类，则创建的类是旧式类。
# 新式类的定义（Python 3.x 中的所有类都是新式类）
class NewStyleClass(object):
    pass

# 继承自 object 的类是新式类
class AnotherNewStyleClass(NewStyleClass):
    pass

# 旧式类的定义（在 Python 2.x 中，未显式继承自 object 的类是旧式类）
class OldStyleClass:
    pass

# 旧式类的定义（在 Python 2.x 中，未显式继承自 object 的类是旧式类）
class AnotherOldStyleClass:
    pass

注意:

后续代码都是些新式类形式

二.对象的创建

1.实例化对象

实例化是指根据类创建类的实例(对象)的过程。

python中，可以根据已经定义的类去创建出一个或多个对象。

结构:

对象名 = 类名()

demo:

class Person(object):
    def __init__(self, name, age):
        self.name = name
        self.age = age

# 创建一个 Person 对象
person1 = Person("Alice", 30)

# 访问对象的属性
print(person1.name)  # 输出：Alice
print(person1.age)   # 输出：30

2.添加对象的属性

结构:

对象.需要添加的属性名 = 需要添加的属性值

demo:

class Person(object):
    def __init__(self, name, age):
        self.name = name
        self.age = age

# 创建一个 Person 对象
person1 = Person("Alice", 30)

# 添加对象属性
person1.city = "New York"
person1.gender = "Female"

# 访问对象的属性
print(person1.name)   # 输出：Alice
print(person1.age)    # 输出：30
print(person1.city)   # 输出：New York
print(person1.gender) # 输出：Female

3.获取对象的属性

结构:

属性 = 对象.需要获取的属性名

demo:

class Person(object):
    def __init__(self, name, age):
        self.name = name
        self.age = age

# 创建一个 Person 对象
person1 = Person("Alice", 30)

# 获取对象的属性
name = person1.name
age = person1.age

# 输出对象的属性
print("Name:", name)  # 输出：Name: Alice
print("Age:", age)    # 输出：Age: 30

4.类的内部创建和获取对象的属性

在类的内部，可以通过 self 来获取当前对象的属性，并且可以使用 self 来创建新的对象属性。

结构:

class 类名(object):

    def 方法(self, parameters):
        对象的属性 = self.属性名  # 内部获取对象的属性
        self.要添加的属性名 = 要添加的属性值  # 内部添加对象的属性

demo:

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def get_info(self):
        info = f"Name: {self.name}, Age: {self.age}"
        return info

    def greet(self):
        greeting = f"Hello, my name is {self.name}."
        return greeting

    def add_friend(self, friend_name):
        # 创建一个新的对象属性
        self.friend = friend_name

# 创建一个 Person 对象
person1 = Person("Alice", 30)

# 调用对象的方法来获取属性和方法
info = person1.get_info()
greeting = person1.greet()

# 输出对象的属性信息和方法返回值
print(info)      # 输出：Name: Alice, Age: 30
print(greeting)  # 输出：Hello, my name is Alice.

# 调用 add_friend() 方法来创建新的对象属性
person1.add_friend("Bob")

# 再次调用 get_info() 方法来获取更新后的属性信息
updated_info = person1.get_info()
print(updated_info)  # 输出：Name: Alice, Age: 30
