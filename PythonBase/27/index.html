Python函数(一)

函数介绍

Posted by     "Toby" on Sunday, June 7, 2020

一.什么是函数？

如果在开发程序时，需要某块代码多次使用，但是为了提高编写的效率以及代码的重用，所以把具有独立功能的代码块组织为一个小模块，这个小模块就是函数

二.函数定义和调用

1.函数定义

作用: 将特定功能的可重复使用的代码封装成代码的过程

结构:

def 函数名:
    代码

demo:

# 定义一个函数，能够完成打印信息的功能
def printInfo():
    print('------------------------------------')
    print('         人生苦短，我用Python')
    print('------------------------------------')

2.调用函数

定义了函数之后，就相当于有了一个具有某些功能的代码，想要让这些代码能够执行，则需要调用它，这个过程就是调用函数

结构:

函数名()

demo:

# 定义完函数后，函数是不会自动执行的，需要调用它才可以
printInfo()

三.函数的文档说明

>>> def test(a,b):
...     """用来完成对2个数求和"""
...     print("%d"%(a+b))
... 
>>> 
>>> test(11,22)
33

如果执行，以下代码

>>> help(test)

能够看到test函数的相关说明

Help on function test in module __main__:

test(a, b)
    用来完成对2个数求和
(END)

小结:

在函数定义之后在函数定义后的那行添加上对函数的注释，使用help函数就可以直接打印函数的注释和函数名
