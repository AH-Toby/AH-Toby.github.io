Python奇技淫巧(二)

Python中集合，列表，元组对比

Posted by     "Toby" on Wednesday, June 10, 2020

一.集合(set)

集合(set)是一种无序、可变的数据类型，用于存储唯一的元素。集合中的元素是无序排列的，并且不能重复。集合通常用于存储无序且不重复的元素，例如去重操作、集合运算等。

结构:

{值1，值2，值3}

特点:

  • 唯一性: 集合中的元素是唯一的，不会重复出现。
  • 无序性: 集合中的元素是无序排列的，无法通过索引来访问集合中的元素，只能通过遍历取值。
  • 可变性: 集合是可变的，你可以向集合中添加或删除元素。

1.集合常见操作

1.1.添加元素

add()方法: 添加单个元素

set_value = (1, 2, 3, 4)
set_value.add(5)

update()方法: 添加多个元素。

set_value = {1, 2, 3, 4}
set_value.update({5,6,7})

1.2.删除元素

remove()|discard()方法: 删除集合中的元素

set_value = {1, 2, 3, 4}
set_value.remove(1)

set_value = {1, 2, 3, 4}
set_value.discard(1)

两者不同的是remove如果删除不存在的元素会报错而discard则不会

pop()方法: 删除任意元素

set_value = {1, 2, 3, 4}
set_value.pop()

一般不使用因为会删除任意一个元素

1.3.集合运算

包括取两个集合元素的并集、交集、差集和对称差集等操作。

# 创建集合
set1 = {1, 2, 3}
set2 = {3, 4, 5}

# 求并集
union_set = set1 | set2  # 或者使用 set1.union(set2)

# 求交集
intersection_set = set1 & set2  # 或者使用 set1.intersection(set2)

# 求差集
difference_set = set1 - set2  # 或者使用 set1.difference(set2)

# 求对称差集
symmetric_difference_set = set1 ^ set2  # 或者使用 set1.symmetric_difference(set2)

二.列表(list)

列表(list)是一种有序、可变的数据类型，用于存储一组元素。列表中的元素可以是任意类型的，并且允许重复。列表是Python中最常用的数据结构之一，用于存储一系列相关的数据。

结构:

[值1，值2，值3]

特点:

  • 有序性: 列表中的元素是有序排列的，可以通过索引来访问列表中的元素。
  • 可变性: 列表是可变的，你可以通过索引来修改列表中的元素，也可以对列表进行添加、删除等操作。
  • 元素类型多样性: 列表中的元素可以是任意类型的，甚至可以包含其他列表或字典等复杂类型。
  • 允许重复: 列表中的元素可以重复出现，同一个元素可以在列表中出现多次。

1.列表常见操作

  • 访问元素: 通过索引来访问列表中的元素，索引从0开始。
  • 修改元素: 通过索引来修改列表中的元素的值。
  • 切片操作: 可以使用切片操作来获取列表的子列表。
  • 添加元素: 使用 append() 方法在列表末尾添加元素，使用 insert() 方法在指定位置插入元素。
  • 删除元素: 使用 remove() 方法删除指定值的元素，使用 pop() 方法删除指定位置的元素。
  • 列表运算: 包括列表的合并、重复、排序等操作。

demo:

# 创建列表
my_list = [1, 2, 3, 4, 5]

# 访问元素
print(my_list[0])  # 输出: 1

# 修改元素
my_list[0] = 10

# 切片操作
sub_list = my_list[1:3]

# 添加元素
my_list.append(6)

# 删除元素
my_list.remove(3)

# 列表运算
new_list = my_list + [7, 8, 9]

三.元组(tuple)

元组（Tuple）是一种有序、不可变的数据类型，用于存储一组元素。元组和列表类似，但是元组的元素不能被修改、添加或删除，一旦创建就无法改变。

结构:

(值1，值2，值3)

特点:

  • 不可变性: 元组是不可变的，一旦创建后，不能对其进行修改。这意味着你无法修改元组中的元素，也无法添加或删除元素。
  • 有序性: 元组中的元素是有序排列的，可以通过索引来访问元组中的元素。元组中的元素可以是任意类型的，包括数字、字符串、列表、元组等。
  • 元素类型多样性: 元组中的元素可以是任意类型的，并且元组中的元素类型可以不相同。
  • 元素重复性: 元组中的元素可以重复出现，同一个元素可以在元组中出现多次。

1.元组的常见操作

  • 访问元素: 可以通过索引来访问元组中的元素，索引从0开始。
  • 切片操作: 可以使用切片操作来获取元组的子元组。
  • 元组运算: 包括元组的合并、重复等操作。
  • 解包(unpacking): 将元组中的元素解包并赋值给多个变量。

demo:

# 创建元组
my_tuple = (1, 2, 3, 4, 5)

# 访问元素
print(my_tuple[0])  # 输出: 1

# 切片操作
sub_tuple = my_tuple[1:3]

# 元组运算
new_tuple = my_tuple + (6, 7, 8)

# 解包
a, b, c = my_tuple[0:3]

四.相互转换

集合元组列表他们之间是可以相互转换的

1.转换为集合

使用 set() 函数可以将列表或元组转换为集合。

demo:

my_list = [1, 2, 3, 4, 5]
my_tuple = (6, 7, 8, 9, 10)

# 列表转换为集合
set_from_list = set(my_list)

# 元组转换为集合
set_from_tuple = set(my_tuple)

注意:

使用set()，可以快速的完成对list中的元素去重复的功能

2.转换为列表

使用 list() 函数可以将集合或元组转换为列表。

demo:

my_set = {1, 2, 3, 4, 5}
my_tuple = (6, 7, 8, 9, 10)

# 集合转换为列表
list_from_set = list(my_set)

# 元组转换为列表
list_from_tuple = list(my_tuple)

3.转换为元组

使用 tuple() 函数可以将集合或列表转换为元组。

demo:

my_set = {1, 2, 3, 4, 5}
my_list = [6, 7, 8, 9, 10]

# 集合转换为元组
tuple_from_set = tuple(my_set)

# 列表转换为元组
tuple_from_list = tuple(my_list)

注意:

这些转换方法可以帮助你在需要不同容器类型之间进行转换时方便地进行操作。请注意，在执行转换时，可能会丢失一些数据信息，例如集合转换为列表或元组时，可能会丢失元素的顺序。
